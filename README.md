# Audio Looping Tool

A simple GUI tool to convert non-looped `.vsnd_c` audio files to looped versions for CS2

## ⚠ Important Warning

**Please save all your unsaved work in Hammer before using this tool. Moving looped audio files may crash your game if qennu poorly programmed.**

**The tool is designed to work with MP3 files only!**


## How to Convert Audio to .vsnd_c Format

1. Make sure your audio `(.mp3)` is located in `...\content\csgo_addons\your_addon\sounds`

2. Open "Asset Browser" and find your audio. It will be in .vsnd format. Right-click on it and press "Recompile" -> "Full"

3. Your `.vsnd_c` will be located in `...\game\csgo_addons\your_addon\sounds`

4. Next you'll make it looped via this program

## How to Use This Tool

1. Press the button and choose `.vsnd_c` file

2. In the next window, select where to save looped `.vsnd_c` file (by default, the suffix "_looped" will be added)

3. Move your saved file to `...\game\csgo_addons\(your_addon)\sounds` and make sure you have created looped soundevent

## Notes:

• The file must be in `.vsnd_c` format
